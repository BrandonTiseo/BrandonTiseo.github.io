Client Project Overview

Project Overview

  1. The overall purpose of this application is to advertise and provide a website for Lite-d-core, a Christmas light installation business based in Long Island, NY.
  2. The website is intended for customers to gather information about Lite-d-core, learn about their services, obtain contact information, and sign up for their light installation service.
  3. The website will include an About page, a FAQ page, a Services page, a Product Catalog page, a page with a form for requesting a quote, and an Image Gallery page.

Dynamic Functionality on the Website

  • Forms (Quote Page) — users can submit their information through a form to request a quote.
  • Photo Gallery (Image Gallery Page) — users can view a photo gallery where they can enlarge or download images.
  • Product Catalog (Services Page) — users can select different products from the catalog, save them to a list, and export or submit that list to the quote form.
